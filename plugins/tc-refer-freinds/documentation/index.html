How to Install

Installing Refer A Friend Is very simple and only takes seconds!

1. In your WordPress dashboard go to your Plugins and add a new plugin using the upload option.

2. Select the tc-refer-friends.zip file NOT the file you downloaded from CodeCanyon. If you select the wrong file you will get an error that the plugin cannot be installed.

3. Activate and you are ready to go! Continue on to the next part of setup.

Setting Up Pages

Before your users can start referring sales they need to be able to find ther ID number used for tracking sales.

Create a new page titled Refer A Friend or whatever you want to use. For the page content just enter this shortcode: 

[tcraf-refer-page]

Then click save!

 

All of the information will be generated for the page by the shortcode, you do not need to add any content unless desired. If you visit the page you will see it gives a description of the Refer A Friend system, gives the user information about their referral links, shows referral stats, and also lists coupons they have earned and not used yet.

Note: The shortcode will also automatically check to make sure the user is logged in to your site before continuing so you do not need to take this step. (This is required to track the sales and coupons.)

Settings / Options

Under the WooCommerce Tabs in your dashboard you will see a new page titled 'Refer A Friend.' This is where you can setup all of your options for the RAF system. 

 

General Settings

Enable / Disable RAF - Here you can quickly enable and disable the plugin for testing without having to use the plugin activator.

Buyer Discount Setup - Here you can enable the plugin that a buyer gets when they are referred. Simply choose to enable the feature and then select the coupon you want to apply.

Cart / Success Message - Choose the message that will be displayed on the cart / checkout pages when a buyer has been given the "Refer a Friend" discount..

 

Referral Settings

Enable Referral Coupon - If enabled, your referring user will also get a discount for referring a sale to your site. This is a one time use coupon tied to the user. 

Minimum Cart Amount - Here you can set a minimum purchase amount for the referring user to get their coupon / referral. You can set this to 0 to disable.

Referral Coupon Type - Choose the type of coupon to use percent or flat rate discount.

Coupon Amount - Enter the amount to use for the discount. For instance if you are using flat rate "5.50" or percent: "5"

Coupon Email Subject - Enter the subject of the email that will be sent notifiying the user of their coupon.

Coupon Email Message - Here you can enter the custom email message that will be sent to the user when they make a referral and get a coupon from it. The actual coupon code will be inserted underneath your message.

Using The Plugin

Users can refer people to any page of your WordPress site to track sales. This makes it easy to link to product pages, deal pages, homepages, etc. allowing sales to come from anywhere!

After a user completes a sale made via a referral the session is closed and the users sales will no longer apply to the refer a friend system (unless they are referred again.) 

What makes this plugin different from an affiliate system is that the value being generated is done all within the site without having to pay out money for generating sales. You don't need to worry about payment processors and tracking fradulent sales as the coupon system prevents all of these issues. It's easy, friendly, and users love it!

Dashboard Widget - The plugin also comes with a custom Dashboard widget that will so you the overall stats of your referral system included cool stats like the total value of sales generated by Refer A Friend. The widget will also display a short list of recent referrals made on the site.
